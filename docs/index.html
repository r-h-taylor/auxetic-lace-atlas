Interactive visualizer

The interactive visualizer is under construction. When complete, it will let you:

  • Browse all 321 grounds organized by hole-shape family
  • See pair-diagram and thread-sketch swatches for each topology
  • Toggle between spring (pin-jointed) and beam (Euler-Bernoulli) mechanical models
  • Slide through the parameter sweep (kang for spring, aspect ratio for beam)
  • Apply uniaxial tension at any angle and see the deformed configuration animate
  • Plot ν(θ) on a polar diagram, with νmin and νmax highlighted
  • Submit your own topologies and see how they rank against the catalog

For now, you can:

  • Browse the source on GitHub
  • Download the raw atlas.json and thumbnails/
  • Read the README for the full data model and pipeline
